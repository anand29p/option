Setup From Scratch
------------------
Use Anaconda / Miniconda Prompt, or a PowerShell where conda is available.

1. Go to the project folder:

   cd C:\genai\option

2. Create the conda environment:

   conda create -n option_bot_312 python=3.12

3. Activate it:

   conda activate option_bot_312

4. Install dependencies:

   pip install -r requirements.txt

5. Optional but recommended for news sentiment:

   python -c "import nltk; nltk.download('vader_lexicon')"

6. Create .env from the example if needed:

   copy .env.example .env

7. Edit .env and set at least:

   MODE=paper
   PAPER_CAPITAL=100000
   MAX_CAPITAL_PER_TRADE=10000
   ACTIVE_STRATEGY_ALLOWLIST=FINNIFTY:RSIDivergence
   SHADOW_SIGNAL_LOG=true
   SHADOW_LOG_FILE=logs/shadow_signals.csv
   DHAN_CLIENT_ID=your_client_id
   DHAN_ACCESS_TOKEN=your_access_token
